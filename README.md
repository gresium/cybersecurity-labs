# Cybersecurity Labs 

A collection of hands-on cybersecurity projects built during my **Simplilearn x IIT Kanpur alumni program**, combined with independent lab work.

This repository documents my learning journey through **practical security exercises**, focusing on understanding how attacks work and how systems can be defended in real-world scenarios.

## About This Repository

The goal of this repository is simple:
- Learn by **doing**
- Practice in **safe, legal lab environments**
- Build a solid foundation across multiple cybersecurity domains

Each project reflects topics covered in a structured cybersecurity curriculum and is documented to show **process, findings, and takeaways**, not just results.

## Example Projects

- **Minimal Honeypot**  
  Implemented a lightweight honeypot to capture unauthorized access attempts and analyze attacker behavior through logs.

- **SSH Brute-Force Analysis**  
  Studied real brute-force patterns against an SSH service to better understand common attack techniques.

- **Vulnerability Assessment Labs**  
  Practiced identifying and evaluating vulnerabilities in controlled environments.

- **Reconnaissance & Information Gathering**  
  Explored how exposed services, metadata, and digital footprints can be discovered.
